## How to install
Make sure you have python 3.7 installed and grip uses an older version of grip

    git clone http://github.com/frc5024/DeepVision
    cd DeepVision
    pip install -r requirements.txt
    python DeepVision
    
    {BOOM SERVER RUN!!}
